# Quoterback
A React Native mobile application for delivering inspiring quotes.

## Planned Features

- **Beautiful Quote Display**: Quotes on curated background images
- **Mood-Based Selection**: Get quotes based on your current emotional state
- **Smart Filtering**: Search by author, topic, category, and mood
- **Favorites**: Save and organize favorite quotes
- **Daily Notifications**: Customizable delivery time and frequency
- **Offline Support**: Work without internet connection
- **Quote Reporting**: Community-driven quality control

## Tech Stack

- React Native with Expo
- Zustand for state management
